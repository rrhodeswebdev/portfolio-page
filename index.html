Link Brain

Link Brain is built for SEO professionals to have a tool that can help them manage all of their contacts for backlink building campaigns. The application comes with a simple dashboard to give users a general overview of their outreach success through visual graphs and cards. Link Brain also allows users to build campaigns to house contacts, and allow the contacts to be sorted by multiple data points.

Hiker Helper

An application to help people find hiking trails in a specific area. This application allows users to input any area and the app will generate a list of trails within a 25 mile radius and provide a 5-day weather forecast. Utilizing geocode technology, Google Maps, The Hiking Project API, and the OpenWeatherMap API.
